Projet Modélisation et bases de données

Découvrir la modélisation à travers le développement d'une base de données.

Présentation du Projet

Ce projet a pour thématique les bases de données relationnelles. Il permet de découvrir les rudiments de la modélisation à travers le développement d'une base de données.

Thèmes abordés :

  • SI et données
  • Merise
  • Algèbre relationnelle
  • Bases de données et SQL
  • Requêtes SQL
  • LDD / Transactions / Procédures

Le projet consiste à réaliser une base de données pour le ministère de l'écologie, afin de stocker et interroger des données publiques sur la qualité de l’air dans les grandes villes de France.
